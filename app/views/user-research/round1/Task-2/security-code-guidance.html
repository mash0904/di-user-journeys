How to get a security code

When you create your GOV.UK One Login, you can choose to get security codes by text message or authenticator app.

To get security codes by text message to a non-UK mobile phone number, select ‘I do not have a UK mobile phone number’ when you are asked to enter a phone number.

If you do not receive a security code by text message

If you’re having trouble getting your security code, you can try having the code sent again by selecting the ‘send the code again’ link.

If you are creating an account, you should check you have entered the phone number correctly and enter it again if not. You can do this by selecting the ‘use a different phone number’ link.

If you still do not receive the code, it could be because your phone carrier is not able to deliver the text message containing the security code.

This can be because:

  • your phone has no reception
  • the sender of the text message has been blocked on your device

Try creating a GOV.UK One Login or signing in again. If you’re creating a GOV.UK One Login, make sure you enter your phone number correctly.

If this does not work, it’s worth checking if this number has been blocked as a sender on your device.

You can also try using an authenticator app to get a security code instead of a text message.

If you do not receive a security code by email

Try creating a GOV.UK One Login again at: gov.uk/account.

Make sure you enter your email address correctly and check your spam or junk folders.

If you are using a work or business email, check with your IT team to make sure that emails from GOV.UK One Login are not being blocked or restricted.

If the code does not arrive in a few minutes, select the ‘Send the code again’ link, in case there was an error.

If your code does not work

There are a few reasons why the code you are entering does not work. It could be because:

  • you did not enter it correctly
  • you requested more than one code, and the code you are entering is not the most recent code

You may need to wait a short while and then try creating a GOV.UK One Login or signing in again.

Check you have entered the code correctly, and wait a few minutes before requesting a new code.
